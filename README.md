House Price Prediction – Model Validation & Hyperparameter Tuning

This repository contains Task 3 of my AI & Machine Learning Internship, where I enhanced a House Price Prediction model by applying professional model validation and optimization techniques.

The project focuses on improving model reliability through overfitting detection, 5-Fold Cross-Validation, and Hyperparameter Tuning using GridSearchCV. Multiple regression models were evaluated and compared using standard performance metrics to identify the best-performing model
